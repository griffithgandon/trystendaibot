## Trystendai bot
Телеграм бот находящийся в активной разработке. Основной библиотекой является [pyTelegramBotAPI](https://pypi.org/project/pyTelegramBotAPI/).
Разрабатывается для панели [3x-ui](https://github.com/MHSanaei/3x-ui), но при желании может быть адаптирован под любую другую панель с похожей API.

В работе за отсутсвием глубоких познаний в используемых библиотеках, используется методика [вайб-кодинга](https://ru.wikipedia.org/wiki/%D0%92%D0%B0%D0%B9%D0%B1-%D0%BA%D0%BE%D0%B4%D0%B8%D0%BD%D0%B3).
Но даже так, при этом мы стараемся вчитываться в код и 
понимать вносимые изменения в угоду дальнейшего дебаггинга и самостоятельного расширения функционала
